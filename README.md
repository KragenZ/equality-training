# ⚖️ Equality Training — One Shot!

**Equality Training** is a premium, high-performance command center designed for training hosts in the Roblox game *Ro-Ghoul*. It streamlines the management of players and provides automated script generation for a variety of training events.

## 🚀 Live Site
View the application here: **[https://kragenz.github.io/equality-training/](https://kragenz.github.io/equality-training/)**

## ✨ Features

### 🏆 Tournament 2.0 (Automatic Matchmaking)
- **Random Matchmaking**: Paste a list of player names and randomly pair them for **1v1** or **2v2** matches.
- **Bye Logic**: Automatically handles odd numbers of participants, ensuring everyone gets to play fairly.
- **Dynamic Scripting**: Integrated script lines that update as you generate matchups.

### 💥 TDM 2.0 (Live Score Tracking)
- **Interactive Scoring**: Increment or decrement scores for Team 1 and Team 2 with one click.
- **Reset Capability**: Quickly reset the score for a fresh match.
- **Instant Announcement**: Copyable script lines that automatically call out the current score (e.g., "The score is now 1 - 0!").

### 💀 Juggernaut IQ
- **Life Tracker**: A dedicated counter to keep track of the Juggernaut's **10 lives**.
- **Victory Condition**: Automatically updates script lines when the Juggernaut is defeated.

### 🌿 Hunger Games IQ
- **Side Assignment**: Instantly randomize players into **Ghoul** and **CCG** teams for hiding.
- **Grace Period Integration**: Pre-written lines for grace periods and PVP-On announcements.

### 🎮 Classic Events
- **FFA (Free For All)**, **Gladiators**, **KOTH (King of the Hill)**, and more.
- **One-Click Copy**: High-speed copy buttons for every script line.
- **Permission Abbreviations**: Quick responses for standard Discord permissions (PTA, PTL, PTC, PTR, PTS).

### ✏️ Custom Messages
- Add, save, and manage your own frequently used training messages directly in the app.

## 🛠️ Technology Stack
- **HTML5 / CSS3 / Vanilla JS** (No dependencies, ultra-lightweight)
- **Void Ethereal Theme**: Premium dark-obsidian aesthetic with high-end glassmorphism and subtle animations.
- **Logo**: Custom Inline SVG "Scales of Justice".

## 📖 How to Use
1. **Navigate**: Use the tabs or keyboard shortcuts (1–3) to switch between Abbreviations, Events, and Custom.
2. **Configure**: Select an event (e.g., Tournament) and fill in the configuration fields.
3. **Execute**: Click the copy icon (📋) next to any line or use the "Copy Full Script" button to paste directly into the game.
